Typora 配置图床(使用腾讯云对象存储COS)

前提条件

  1. typora 安装已经成功
  2. 机子上面有node 或者yarn 可以安装 pic-Go

安装步骤

1. 下载 picGo-Core

yarn global add picgo

可以使用 which picgo 找到 picgo 的安装地址

2. 配置腾讯云

2.1 进入腾讯云 -> 对象存储 -> 创建存储桶

2.2 存储桶配置

2.3 密钥管理

控制台->访问管理->API密钥管理->创建子用户（为了安全起见）-> 新建密钥

3. 配置腾讯云cos 到picgo-core

因为picgo默认是支持smms的， 所以我们先运行的命令， 会生成一个smms 的模版

picgo set uploader

然后之后就会在

/Users/kevincheng/.picgo中出现

如下所示

这边使用的是v5版本

Typora 配置

command 是 {node-path} {picgo-path} u

node-path 可以根据 which node 得到

Picgo-path 可以根据which picgo 得到

效果

可以看到已经变到网络地址了
